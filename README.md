# Software component for DuelDash -- Senior Capstone Project
Code base for DuelDash software component. It contains a web application that host a local server for race information to be calculated and stored. It also has a public viewing page that shows real time result accessible through URL.
## Techonology used:
* HTML/CSS
* JavaScript
* Electron.js
* PorchDB
