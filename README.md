Software component for DuelDash -- Senior Capstone Project
Techonology used:
* HTML/CSS
* JavaScript
* Electron.js
* PorchDB

This code base contains a web application that host a local server for race information to be calculated and stored. It also has a public viewing page that shows real time result accessible through url.
